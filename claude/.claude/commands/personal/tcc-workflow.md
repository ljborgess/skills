# 🎓 Skill: TCC Workflow

## Description
Workflow completo para desenvolvimento de TCC do zero até a defesa. Define etapas, entregáveis, prazos sugeridos e critérios de qualidade para cada fase do trabalho acadêmico.

---

## Hard Rules
1. **NUNCA** avance de fase sem completar os entregáveis da fase anterior
2. **SEMPRE** mantenha backup versionado do documento (Git ou Drive)
3. **NUNCA** deixe para revisar ABNT no final — aplique durante a escrita
4. **SEMPRE** registre todas as fontes no momento da leitura
5. **NUNCA** escreva a introdução definitiva antes de ter os resultados

---

## Fases do TCC

### Fase 1: Definição (Semanas 1-2)
- [ ] Tema definido (área + recorte específico)
- [ ] Problema de pesquisa formulado como pergunta
- [ ] Justificativa (relevância prática + teórica)
- [ ] Objetivo geral (1 verbo + 1 objeto + 1 contexto)
- [ ] 3-5 objetivos específicos mensuráveis
- [ ] Viabilidade confirmada (prazo + acesso a dados)

**Critério de saída:** Orientador aprovou o problema e objetivos

### Fase 2: Referencial Teórico (Semanas 3-6)
- [ ] Mapa conceitual dos temas principais
- [ ] Mínimo 20 fontes fichadas
- [ ] Capítulos do referencial redigidos
- [ ] Citações formatadas em ABNT

### Fase 3: Metodologia (Semanas 5-7)
- [ ] Tipo de pesquisa definido e justificado
- [ ] Universo/amostra definidos
- [ ] Instrumento de coleta criado e validado
- [ ] Procedimento de análise descrito

### Fase 4: Coleta e Análise (Semanas 8-12)
- [ ] Dados coletados conforme metodologia
- [ ] Análise realizada
- [ ] Resultados documentados
- [ ] Discussão relacionando resultados ao referencial

### Fase 5: Escrita Final (Semanas 13-15)
- [ ] Introdução revisada (com resultados em mãos)
- [ ] Conclusão escrita (retoma cada objetivo)
- [ ] Revisão ortográfica e gramatical
- [ ] Formatação ABNT completa (NBR 14724, 6023, 10520)
- [ ] Resumo e Abstract finalizados

### Fase 6: Defesa (Semana 16+)
- [ ] Slides (10-15 slides, 20 min de apresentação)
- [ ] Simulação de banca realizada
- [ ] Respostas preparadas para perguntas prováveis

## Template de Fichamento de Fonte

```markdown
## Fichamento: [Título]
**Autor(es):** [Nome(s)] | **Ano:** [ano] | **Tipo:** Artigo/Livro/Dissertação

### Ideia Central
[1-2 frases resumindo o argumento principal]

### Citações Úteis
> "[citação direta]" (AUTOR, ano, p. X)

### Como Usar no TCC
[Em qual seção e para qual argumento esta fonte serve]
```
